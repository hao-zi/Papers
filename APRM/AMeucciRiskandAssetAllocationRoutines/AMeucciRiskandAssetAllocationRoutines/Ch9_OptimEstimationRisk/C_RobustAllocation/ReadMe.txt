This folder includes the SeDuMi package and tutorial materials.
SeDuMi is a freeware optimization software, originally developed by Jos F. Sturm,
that runs in the MATLAB environment. SeDuMi is not associated with www.symmys.com.
The most recent version of the SeDuMi package can be downloaded from http://sedumi.mcmaster.ca
In order to run the optimization script RobSocpMVPortf.m 
add the folder "SeDuMi_1_1" to the MATLAB path.
